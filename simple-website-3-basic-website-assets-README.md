    # Simple Website Series — Project 3: Basic Website Assets

    ## Overview
    This project is part of the **Simple Website Series**, a set of 19 small, incremental examples that introduce core website concepts step by step.

    Introduces separate CSS and JavaScript files so structure, style, and behavior are no longer mixed together.

    ## What You'll Learn
    - How HTML, CSS, and JavaScript work together
    - Why separating files improves clarity
    - A simple button-driven interaction with JavaScript

    ## How to Run
    1. Download or clone this repository
2. Open `index.html` in your browser

No build tools, package installation, or local server are required for the base example.

    ## Project Structure
    ```text
404.html
README.md
index.html
script.js
styles.css
```
